## Decisions Made

- **Validation gate is structural-only**: `passed` flag in quality report is too strict for back-adjusted continuous contracts; checking `duplicate_timestamps`, `negative_volumes`, `inverted_high_low`, `null_required_fields` directly is correct. Large gaps at contract rolls are non-fatal warnings.
- **featureset_hash is blake2b-16**: Short enough for filenames if needed, long enough to be collision-resistant for this use case. Falls back to "unknown" gracefully if registry lookup fails.
- **bollinger prefix map in test**: `compute_bollinger()` returns `bb_*` columns; test maps config name `bollinger` → prefix `bb`. Documented in test comment rather than changing indicator code.
- **Stationarity ADF with maxlag=30**: `autolag="AIC"` on 457K obs is prohibitively slow on this hardware; fixed lag cap is defensible for daily-bar-equivalent dynamics.
- **Recommendation: intraday VWAP MR with extended hold** — OU HL ~165 min is longer than ZN but intraday-tradeable during London/NY overlap (12:00–17:00 UTC); flatten by 21:00 UTC.
