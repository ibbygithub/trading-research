## Decisions made

- **Manual prose describes v1.0 state, not transitional state.** Codified after Ibby's Chapter 5 review pushback. The chapter does not document its own scaffolding. Saved as durable feedback memory.
- **The "21,000 files" figure is wrong** in the storage-pressure narrative because it bundled `.venv/` (~18,600 vendored Python package files) with project artefacts. Real project file count is ~1,700; pressure is GB-driven. Corrected in chapter and memory.
- **`clean clean` → `clean canonical`** for the CLEAN-layer reaping subcommand. The CLEAN layer is the canonical OHLCV form (Chapter 4 §4.1) so the rename is descriptive, not arbitrary.
- **Three-tier trial registry** (live, compact, delete) instead of two-tier (live, archive). Validation-mode trials never reach the deletion tier at any age. The deletion tier requires both the age threshold and the absence of any live `parent_sweep_id` reference, to prevent orphaning a sweep's lineage. DSR consults both live and compacted archive so the deflation count stays correct.
