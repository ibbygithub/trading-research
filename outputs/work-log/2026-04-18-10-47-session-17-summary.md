## Decisions Made

- PSR/DSR formula is correct for Pearson kurtosis (normal=3), not excess/Fisher (normal=0). Marked as Severity-2 finding (caveat, not outright bug in the formula itself) because the formula is documented correctly — the risk is in callers using scipy default `kurtosis()` without `fisher=False`. Follow-up session needed to grep and fix callers.
- Classifier purge changed from symmetric KFold (which includes post-test data in training) to strict walk-forward: train always on `[0, val_start - purge_bars)`. First folds may be skipped if insufficient training data.
- walkforward.py `gap_bars` + `embargo_bars` interpreted as: gap excludes bars between folds; embargo excludes bars at the start of each fold (except fold 0). ValueError raised if parameters consume too much of the dataset.
- Meta-label Calmar: if entry_ts/exit_ts columns are absent, falls back to len(df) as span_days. Documented in code comment.
- FOMC dates: used statement release dates (last day of the 2-day meeting), which is the day markets move. Emergency actions included because they produce the same price impact.
