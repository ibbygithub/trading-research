## Decisions Made

- **Inclusive bar counts in drawdowns**: `duration_bars = trough_i - start_i + 1` to match intuitive "number of bars in the phase" convention
- **NaN as +inf in degradation check**: When historical Calmar is NaN (no drawdown, perfect period), treat it as +inf so any finite losing recent period correctly triggers the degradation flag
- **Default to v2**: `generate_report()` defaults to `version="v2"` — v1 template still accessible by passing `version="v1"` explicitly
- **Trials registry on filesystem**: `runs/.trials.json` — simple append-only JSON, no database. Counts all entries for a trial_group as n_trials for DSR
- **scipy installed via pip into venv**: uv failed to reinstall due to locked trading-research.exe process; used `ensurepip` + `pip install scipy` directly. Needs `uv add scipy` committed once the lock clears
- **Monte Carlo n_iter=1000 by default in report**: balances runtime (~2s) with distribution quality on 11,887 trades
